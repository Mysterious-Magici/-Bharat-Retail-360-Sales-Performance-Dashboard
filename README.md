🇮🇳 Bharat-Retail 360 – Sales Performance Dashboard

Bharat-Retail 360 is an interactive retail analytics dashboard that provides a comprehensive overview of sales performance across products, customers, regions, and time. It helps businesses analyze trends, identify top-performing segments, and make data-driven decisions.
